# TypeScript Express

REST API with the following technologies:

- TypeScript
- Node.js
- Express
- Mongoose
- Jest

## Running the app

```
# install dependencies
npm i

# run in dev mode on port 3000
npm run dev

# generate production build
npm run build

# run generated content in dist folder on port 3000
npm run start
```

## Testing

### Jest with supertest

```
npm run test
```

### Use watch

```
npm run test-watch
```
